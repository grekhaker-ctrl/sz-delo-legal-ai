## 📱 ПОДРОБНО ПО РАЗДЕЛАМ

### 1. 💬 ЧАТ С ИИ-ЮРИСТОМ

**Для чего:** Быстрые юридические консультации

**Как использовать:**

1. Откройте раздел «Чат с ИИ-юристом»
2. Введите вопрос в поле внизу
3. Нажмите Enter
4. Получите ответ

**Примеры вопросов:**

✅ **Можно спрашивать:**
- «Что такое неустойка по договору подряда?»
- «Какие риски в договоре субподряда?»
- «Кто отвечает за скрытые дефекты?»
- «Можно ли прописать штраф 10% от цены договора?»
- «Какой гарантийный срок в строительстве?»

❌ **Нельзя спрашивать:**
- «Какая погода в Москве?»
- «Напиши письмо контрагенту»
- «Позвони юристу»

**Важно:** Бот отвечает **только на юридические вопросы**!

---

### 2. 📄 АНАЛИЗ РИСКОВ ДОГОВОРА

**Для чего:** Проверка договора перед подписанием

**Как использовать:**

1. Откройте раздел «Анализ рисков договора»
2. Перетащите файл договора в область загрузки
   - Поддерживаются: PDF, DOCX, TXT
   - Максимальный размер: 10 MB
3. Нажмите кнопку **«🔍 Начать анализ рисков»**
4. Дождитесь результата (30-60 секунд)
5. Изучите отчёт

**Что показывает отчёт:**

```
┌─────────────────────────────────────┐
│  ⚠️ Можно подписывать, но с правками │
├─────────────────────────────────────┤
│  Уровень риска: 6/10                │
│                                     │
│  Найденные риски:                   │
│  1. Чрезмерная неустойка (1%/день)  │
│  2. Неограниченная ответственность  │
│  3. Нет порядка приёмки работ       │
│                                     │
│  Рекомендации:                      │
│  - Снизить неустойку до 0.1%/день   │
│  - Ограничить ответственность 50%   │
│  - Добавить раздел о приёмке        │
└─────────────────────────────────────┘
```

**Цветовая индикация:**

- 🟩 **Зелёный** — низкий риск, можно подписывать
- 🟨 **Жёлтый** — средний риск, внимание
- 🟧 **Оранжевый** — высокий риск, нужны правки
- 🟥 **Красный** — критично, не подписывать

**Совет:** Всегда проверяйте риски с уровнем «высокий» и «критичный»!

---

### 3. ⚖️ ЮРИДИЧЕСКОЕ ЗАКЛЮЧЕНИЕ

**Для чего:** Официальное мнение по договору

**Как использовать:**

1. Откройте раздел «Юридическое заключение»
2. Загрузите файл договора
3. Нажмите **«📝 Получить заключение»**
4. Получите развёрнутое заключение

**Структура заключения:**

```
1. ТИП ДОГОВОРА
   Договор строительного подряда

2. ЗАКЛЮЧЕНИЕ
   ⚠️ Можно подписывать, но с правками

3. КЛЮЧЕВЫЕ РИСКИ
   - Чрезмерная неустойка
   - Неограниченная ответственность
   - Отсутствие гарантийного срока

4. РЕКОМЕНДАЦИИ
   - Пункт 3.3: Снизить неустойку до 0.1%
   - Пункт 4.3: Ограничить ответственность
   - Добавить раздел о гарантиях
```

**Когда использовать:**
- Договор на крупную сумму (>10 млн ₽)
- Новый контрагент
- Нестандартные условия
- Для руководства компании

---

### 4. 📝 ЗАПОЛНИТЬ ШАБЛОН

**Для чего:** Быстрое создание договора по шаблону

**Как использовать:**

1. Откройте раздел «Заполнить шаблон»
2. Выберите шаблон из списка:
   - Договор строительного подряда
   - Договор субподряда
   - Договор поставки материалов
   - Договор аренды спецтехники
   - Договор на проектные работы
3. Заполните реквизиты:
   - Номер и дата договора
   - Заказчик (название, ИНН)
   - Подрядчик (название, ИНН)
   - Адрес объекта
   - Цена договора
   - Описание работ
4. Нажмите **«📄 Заполнить шаблон»**
5. Скачайте готовый файл DOCX

**Пример заполнения:**

```
Номер договора: 123/2025
Дата: 15.01.2025
Заказчик: ООО "СЗ Дело"
ИНН Заказчика: 7701234567
Подрядчик: ООО "СтройМонтаж"
ИНН Подрядчика: 7709876543
Адрес объекта: г. Москва, ул. Строителей, д. 10
Цена договора: 50000000
Описание работ: Строительство жилого комплекса
```

**Экономия времени:** 30 минут → 2 минуты!

### 5. 🔄 СРАВНИТЬ ВЕРСИИ

**Для чего:** Понять, что изменил контрагент

**Как использовать:**

1. Откройте раздел «Сравнить версии»
2. Загрузите два файла:
   - **Версия 1:** Ваша исходная редакция
   - **Версия 2:** Редакция контрагента
3. Нажмите **«🔍 Сравнить версии»**
4. Изучите результаты

**Что показывает:**

- 🟩 **Зелёный фон** — добавленный текст
- 🟥 **Красный фон** — удалённый текст
- 📊 **Визуальный diff** — наглядное сравнение
- ⚖️ **Юридический анализ** — последствия изменений

**Пример:**

```
Изменения в пункте 3.2:

Было: «Неустойка 0.1% за каждый день просрочки»
Стало: «Неустойка 1% за каждый день просрочки»

⚠️ Юридические последствия:
- Увеличение ответственности в 10 раз
- Риск: ВЫСОКИЙ
- Рекомендация: Отклонить правку
```

**Когда использовать:**
- Контрагент вернул договор со правками
- Нужно понять, что изменилось
- Оценка рисков изменений

- **Внутренняя база СЗ Дело** — типовые договоры, регламенты

**Примеры запросов:**
- «неустойка по договору подряда»
- «гарантийный срок в строительстве»
- «приёмка работ КС-2 КС-11»
- «ответственность субподрядчика»

**Результат поиска:**

```
Найдено в Гарант:

ГК РФ Статья 709. Цена работы
1. В договоре подряда указываются цена подлежащей выполнению работы...

ГК РФ Статья 711. Порядок оплаты работы
```

## 💡 СОВЕТЫ ПО ИСПОЛЬЗОВАНИЮ

### Общие рекомендации:

1. ✅ **Всегда проверяйте критические риски**
   - Обращайте внимание на красные и оранжевые индикаторы
   - Не игнорируйте рекомендации ИИ

2. ✅ **Используйте ИИ как помощника, а не замену**
   - ИИ даёт рекомендации, но решение за вами
   - Всегда проверяйте важные выводы

3. ✅ **Загружайте договоры в хорошем качестве**
   - Для PDF: текст, а не скан
   - Для сканов: чёткое изображение, 300 DPI

4. ✅ **Сохраняйте отчёты**
   - Скачивайте результаты анализа
   - Прикладывайте к договору в архив

5. ✅ **Обучайте коллег**
   - Поделитесь инструкцией
   - Проведите демонстрацию

## ⚠️ ОГРАНИЧЕНИЯ

### Что система НЕ умеет:

❌ **Не отвечает на неюридические вопросы**
- «Какая погода?»
- «Напиши код»
- «Составь письмо»

❌ **Не работает с защищёнными паролем файлами**
- Снимите защиту перед загрузкой

❌ **Не читает рукописный текст**
- Только печатный текст

❌ **Не заменяет юриста**
- Это помощник, а не замена специалиста

### Технические ограничения:

- Максимальный размер файла: 10 MB
- Максимум 10 сообщений в сессии
- Время анализа: 30-60 секунд

## 🆘 ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК

### «LLM недоступен»

**Причина:** Проблема с подключением к YandexGPT/GigaChat

**Что делать:**
1. Проверьте интернет-соединение
2. Сообщите в IT-отдел
3. Используйте резервный режим (базовый анализ)

### «Не могу прочитать файл»

**Причина:** Файл повреждён или формат не поддерживается

**Что делать:**
1. Проверьте формат (PDF, DOCX, TXT)
2. Убедитесь что файл не защищён паролем
3. Попробуйте конвертировать в PDF

### «Слишком долго отвечает»

**Причина:** Большая нагрузка на сервер

**Что делать:**
1. Подождите 1-2 минуты
2. Обновите страницу
3. Попробуйте позже
