# ⚡ БЫСТРЫЙ СТАРТ | СЗ ДЕЛО ЮРИДИЧЕСКИЙ ИИ-АГЕНТ

**Запуск за 5 минут**

---

## 1️⃣ УСТАНОВКА (2 минуты)

```bash
# Проверка Python (нужен 3.11+)
python --version

# Установка зависимостей
pip install -r requirements.txt
```

**Если ошибка с pytesseract:**
```bash
# Windows: скачайте Tesseract OCR
# https://github.com/UB-Mannheim/tesseract/wiki
# Добавьте в PATH
```

---

## 2️⃣ НАСТРОЙКА (2 минуты)

```bash
# Копирование конфига
copy .env.example .env

# Откройте .env в блокноте
notepad .env
```

**Заполните:**

```env
# YandexGPT (получить: https://cloud.yandex.ru/)
YANDEX_FOLDER_ID=b1g...
YANDEX_API_KEY=AQVN...

# ИЛИ только GigaChat (бесплатно)
GIGACHAT_CREDENTIALS=base64_string
```

**Где взять ключи:**

### YandexGPT (платно, ~0.5₽/1000 токенов):
1. https://cloud.yandex.ru/
2. Создайте аккаунт
3. Создайте сервисный аккаунт
4. Выдайте роль `ai.languageModels.user`
5. Создайте API-ключ
6. Скопируйте folder_id и api_key

### GigaChat (бесплатно в тесте):
1. https://developers.sber.ru/
2. Зарегистрируйтесь
3. Создайте приложение
4. Получите client_id и client_secret
5. Закодируйте в base64: `base64(client_id:client_secret)`

---

## 3️⃣ ЗАПУСК (30 секунд)

```bash
# Запуск приложения
streamlit run app.py
```

**Автоматически откроется:** http://localhost:8501

## 4️⃣ ПЕРВЫЕ ШАГИ

### Тест 1: Чат с ИИ-юристом

1. Откройте **"💬 Чат с ИИ-юристом"**
2. Введите вопрос:
   ```
   Что такое неустойка по договору подряда?
   ```
3. Получите ответ

### Тест 2: Анализ рисков

1. Откройте **"📄 Анализ рисков договора"**
2. Загрузите любой договор (PDF или DOCX)
3. Нажмите **"Начать анализ рисков"**
4. Получите отчёт с рисками

### Тест 3: Заполнение шаблона

1. Откройте **"📝 Заполнить шаблон"**
2. Выберите шаблон
3. Заполните реквизиты
4. Скачайте готовый договор

## ⚠️ ВОЗМОЖНЫЕ ПРОБЛЕМЫ

### "LLM недоступен"

**Решение:**
```bash
# Проверьте .env
cat .env

# Убедитесь что ключи заполнены
# Перезапустите приложение
```

### "ModuleNotFoundError"

**Решение:**
```bash
# Переустановите зависимости
pip install -r requirements.txt --upgrade
```

### "Port already in use"

**Решение:**
```bash
# Закройте другое приложение на порту 8501
# Или запустите на другом порту:
streamlit run app.py --server.port 8502
```
